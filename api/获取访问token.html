请求方式

GET 'https://cdpma.baiyaodajiankang.com/openapi/application/token'

请求参数

Headers：

名称 值 是否必须 备注
Content-Type application/json 是 类型
token 访问秘钥 是 appKey和secret获取的访问秘钥

QUERY参数

名称 值 是否必须 备注
appKey 关键字 是 appKey
appSecret 关键字 是 appSecret

返回结果

名称 类型 参数值 是否必须 备注
code int 返回code 必须 0表示正常返回,-1表示请求异常
msg String 返回消息提示 是 返回消息提示
data object 返回的消息体 是 返回的消息体
accessToken string 访问token 是 访问token
expireTime long token过期时间 是 token过期时间
