开发前必读

接入指南

智慧营销平台让您能够通过编程的方式与平台交互。使用Open API来做客户、客户行为、客户标签、客户群组、订单等数据的接入和导出。

创建应用

在开始使用REST API之前需要先创建一个应用，以获取appId和secret。具体请参考应用接入方式。 创建应用后，妥善保管应用ID（appKey）和应用密钥（appSecret）。
