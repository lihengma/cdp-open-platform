群组管理

群组介绍

静态群组

可以通过api或系统界面实时添加客户到群组，且系统不会自动重算的群组我们称为静态群组。

智能群组

不可以通过api或系统界面实时添加客户到群组，但可以根据所配置的规则定时离线计算的群组我们称为智能群组。
